## Tiny PHP Tool to create (anonymous) Surveys
### Features
* Create Surveys, provide answer options, required number of votes and email adress.
* To access a survey, enter the pin provided by survey starter.
* Results will be accessible as soon as the required number of votes are given or after 72h. Organizer will be notified through mail.
